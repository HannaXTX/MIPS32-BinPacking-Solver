# MIPS32-BinPacking-Solver

This project implements a solution to the **Bin Packing Problem** using **First Fit** and **Best Fit** heuristics in **MIPS32 Assembly Language**. It simulates how items are packed into bins to minimize the total number of bins used.

## 🧠 Problem Description

> Given a set of items with varying sizes and a fixed bin capacity, pack the items into the minimum number of bins without exceeding the bin capacity.

This solver uses:
- **First Fit (FF)**: Places each item into the first bin where it fits.
- **Best Fit (BF)**: Places each item into the tightest fitting bin.
